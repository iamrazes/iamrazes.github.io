Introduction

Mufu berasal dari kata "Multi" dan "Fungsi" atau Multifungsi, merupakan sebuah aplikasi android yang dikembangkan pada mata kuliah Mobile Programming II.

Repository : android-mufu

Programming Language : Kotlin

IDE : Android Studio

Tugas 1

Membuat halaman SplashScreen dengan kondisi cek internet, jika ada koneksi internet lanjut ke halaman login, apabila tidak ada internet muncul dialog "Bad Connection, Try Again!"
Membuat halaman Login dengan data di model user, jika salah muncul dialog "Wrong username or password" dengan maximal salah 3x muncul "Reset Password"
Membuat Halaman Menu dan Menampilkan Halaman Menu

Ditugaskan pada tanggal : 13-02-2024
Dipublish pada branch : tugas_1

Hasil :

Splash Screen
Splash Screen muncul dialog ketika tidak ada koneksi internet
Login
Login ketika salah username atau password muncul attempt alert
Login ketika gagal setelah 3 percobaan mnuncul alert reset password
Menu dashboard setelah login
